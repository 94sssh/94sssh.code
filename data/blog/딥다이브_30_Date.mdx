## 30.1. Date 생성자 함수

> 날짜와 시간을 위한 메서드를 제공하는 빌트인 객체이면서 생성자 함수

`Date`는 생성자 함수로, `Date` 생성자 함수로 생성한 `Date` 객체는 현재 날짜와 시간을 나타내는 정수값을 가진다.

### 30.1.1. new Date()

`Date` 생성자 함수를 인수 없이 `new` 연산자와 함께 호출하면 현재 날짜와 시간을 갖는 `Date` 객체를 리턴한다.  
`new` 연산자 없이 호출하면 `Date` 객체가 아닌 날짜와 시간 정보를 나타내는 문자열을 리턴한다.

```js
new Date() // Sun Aug 11 2024 22:02:17 GMT+0900 (한국 표준시) {}
Date() // 'Sun Aug 11 2024 22:03:18 GMT+0900 (한국 표준시)'
```

### 30.1.2. new Date(milliseconds)

`Date`에 숫자 타입의 밀리초를 인수로 전달하면 1970년 1월 1일 00:00:00(UTC)를 기점으로 전달된 밀리초만큼 경과한 날짜와 시간을 나타내는 `Date` 객체를 리턴한다.

```js
new Date(0) // Thu Jan 01 1970 09:00:00 GMT+0900 (한국 표준시) {}
new Date(86400000) // Fri Jan 02 1970 09:00:00 GMT+0900 (한국 표준시) {}
```

### 30.1.3. new Date(dateString)

`Date`에 날짜와 시간을 나타내는 문자열을 인수로 전달하면 지정된 날짜와 시간을 나타내는 `Date` 객체를 리턴한다.

```js
new Date('May 26, 2024 10:00:00') // Sun May 26 2024 10:00:00 GMT+0900 (한국 표준시) {}
new Date('2024/05/26/10:00:00') // Sun May 26 2024 10:00:00 GMT+0900 (한국 표준시) {}
```

### 30.1.4. new Date(year, month[,day,hour,minute,second,millisecond])

`Date`에 연, 월, 일, 시, 분, 초, 밀리초를 의미하는 숫자를 전달하면 지정된 날짜와 시간을 나타내는 `Date` 객체로 리턴한다. 이때 연, 월은 반드시 지정해야 한다.

| 인수        | 내용                                                              |
| ----------- | ----------------------------------------------------------------- |
| year        | 연을 나타내는 1900년 이후의 정수, 0부터 99는 1900부터 1999로 처리 |
| month       | 월을 나타내는 0 ~ 11까지의 정수(주의: 0부터 시작, 0 = 1월)        |
| day         | 일을 나타내는 0 ~ 31까지의 정수                                   |
| hour        | 시를 나타내는 0 ~ 23까지의 정수                                   |
| minute      | 분을 나타내는 0 ~ 59까지의 정수                                   |
| second      | 초를 나타내는 0 ~ 59까지의 정수                                   |
| millisecond | 밀리초를 나타내는 0 ~ 999까지의 정수                              |

## 30.2. Date 메서드

### 30.2.1. Date.now

1970년 1월 1일 00:00:00(UTC)를 기점으로 현재 시간까지 경과한 밀리초를 숫자로 리턴한다.

### 30.2.2. Date.parse

1970년 1월 1일 00:00:00(UTC)를 기점으로 인수로 전달한 시간까지의 밀리초를 숫자로 리턴한다. 인수와 동일한 형식으로 리턴한다.

### 30.2.3. Date.UTC

1970년 1월 1일 00:00:00(UTC)를 기점으로 인수로 전달한 시간까지의 밀리초를 숫자로 리턴한다. `Date.UTC`는 로컬 타임이 아닌 UTC로 인식되며, `new Date(year, month[,day,hour,minute,second,millisecond])`와 같은 형식의 인수를 사용해야 한다.

### get 메서드

- **getFullyear**: `Date` 객체의 연도를 나타내는 정수를 리턴한다.
- **getMonth**: 월(0 ~ 11)을 나타내는 정수를 리턴한다. 0이 1월, 11이 12월에 해당한다.
- **getDate**: 날짜(1 ~ 31)를 나타내는 정수를 리턴한다.
- **getDay**: 요일(0 ~ 6)을 나타내는 정수를 리턴한다. 0이 일요일, 6이 토요일에 해당한다.
- **getHours**: 시간(0 ~ 23)을 나타내는 정수를 리턴한다.
- **getMinutes**: 분(0 ~ 59)를 나타내는 정수를 리턴한다.
- **getSeconds**: 초(0 ~ 59)를 나타내는 정수를 리턴한다.
- **getMilliseconds**: 밀리초(0 ~ 999)를 나타내는 정수를 리턴한다.
- **getTime**: 1970년 1월 1일 00:00:00(UTC)를 기점으로 Date 객의 시간까지 경과된 밀리초를 리턴한다.
- **getTimezoneOffset**: UTC와 Date 객체에 지정된 locale 시간과의 차이를 분 단위로 리턴한다.

### set 메서드

- **setFullyear**: `Date` 객체에 연도를 나타내는 정수를 설정한다. 연도 이외에 월, 일도 설정할 수 있다.
- **setMonth**: 월(0 ~ 11)을 나타내는 정수를 설정한다.
- **setDate**: 날짜(1 ~ 31)를 나타내는 정수를 설정한다.
- **setDay**: 요일(0 ~ 6)을 나타내는 정수를 설정한다.
- **setHours**: 시간(0 ~ 23)을 나타내는 정수를 설정한다. 분, 초, 밀리초도 설정할 수 있다.
- **setMinutes**: 분(0 ~ 59)를 나타내는 정수를 설정한다. 초, 밀리초도 설정할 수 있다.
- **setSeconds**: 초(0 ~ 59)를 나타내는 정수를 설정한다. 밀리초도 설정할 수 있다.
- **setMilliseconds**: 밀리초(0 ~ 999)를 나타내는 정수를 설정한다.
- **setTime**: Date 객체에 1970년 1월 1일 00:00:00(UTC)를 기점으로 경과된 밀리초를 설정한다.

### 30.2.22 Date.prototype.toDateString

Date 객체의 날짜를 문자열로 리턴한다.

### 30.2.23 Date.prototype.toTimeString

Date 객체의 시간을 문자열로 리턴한다.

### 30.2.24 Date.prototype.toISOString

Date 객체의 날짜와 시간을 [ISO 8601](https://ko.wikipedia.org/wiki/ISO_8601) 형식으로 나타낸 문자열을 리턴한다.

### 30.2.25 Date.prototype.toLocaleString

인수로 전달한 Locale을 기준으로 Date 객체의 날짜와 시간을 표현한 문자열을 리턴한다. 인수를 생략하면 브라우저의 시스템 Locale을 적용한다.

### 30.2.26 Date.prototype.toLocalsTimeString

인수로 전달한 Locale을 기준으로 Date 객체의 시간을 표현한 문자열을 리턴한다. 인수를 생략하면 브라우저의 시스템 Locale을 적용한다.
