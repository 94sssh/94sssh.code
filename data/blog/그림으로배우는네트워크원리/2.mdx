## 네트워크를 만드는 것

### 네트워크의 규모는 각각 다르다

구름 모양의 클라우드 아이콘을 통해 네트워크를 표현.  
가정 내 네트워크, 기업 네트워크 등 전후 문맥에 따라 클라우드 아이콘이 가리키는 네트워크 규모가 다름.

### 네트워크를 구성하는 기기

네트워크를 구성하는 구체적인 주요 네트워크 기기 3가지

- 라우터
- 레이어2 스위치
- 레이어3 스위치

데이터 전송 처리 3단계

1. 데이터 수신
2. 데이터를 보낼 곳을 결정
3. 데이터 전송

네트워크 기기의 차이는 두 번째 단계에 있는데, 네트워크 기기의 동작 메커니즘에서는 어느 제어 정보를 참조해 데이터를 보낼 곳을 결정하는지가 중요하다.

### 네트워크의 구체적인 구성

네트워크 기기끼리나 pc, 서버 등을 연결하기 위해 기기에는 인터페이스가 있는데, 가장 일반적인 것은 이더넷 인터페이스.  
인터페이스는 **포트**라고도 불리며, 이더넷 인터페이스는 이더넷 포트나 LAN 포트로도 불림.  
인터페이스끼리 연결하는 것을 링크라고 하며, 인터페이스를 서로 연결하는 케이블을 전송 매체라고 함.  
이렇게 인터페이스를 전송 매체로 연결해 링크를 구성함으로써 네트워크가 형성됨.  
인터페이스는 '0'과 '1'의 디지털 데이터와 물리적인 신호의 경계, 디지털 데이터는 인터페이스에서 송출되어 링크로 전달되어 간다.

### 네트워크를 만든다

현재 LAN을 구축하는 주요 기술은 2가지이다.

- 이더넷
- 무선 LAN(Wi-Fi)

직접 LAN을 구축하려면, 이더넷 인터페이스가 있는 라우터와 레이어2, 레이어3 스위치 등의 네트워크 기기를 준비해야 한다.  
기기들의 네트워크 인터페이스를 LAN 케이블로 연결하면 기기 사이에 링크가 구성되고 유선 LAN이 만들어진다.  
무선 LAN을 이용하려면 무선 LAN 액세스 포인트와 무선 LAN 인터페이스가 있는 기기를 준비하면 된다.

### 어떤 네트워크를 만들고 싶은가?

| 프로세스 | 개요                                                                    |
| :------: | ----------------------------------------------------------------------- |
| 요건정의 | 네트워크에 필요한 기능과 성능을 명확히 한다.                            |
|   설계   | 요건정의에서 결정한 요건을 만족할 수 있도록 구체적인 네트워크 구성 결정 |
|   구축   | 설계한 네트워크를 바탕으로 기기의 배치, 배선, 필요한 설정을 한다        |
| 운용관리 | 네트워크가 정상 동작하도록 점섬 및 문제 복구                            |

### 네트워크 구성을 파악하자

네트워크를 설계하기 위해서는 네트워크 구성도인 `논리구성도`와 `물리구성도`를 파악해 두는 것이 중요하다.

- 논리구성도  
  네트워크끼리 어떻게 연결되는지를 나타낸 구성도.  
  하나의 네트워크는 라우터 또는 레이어3 스위치로 구분되고, 라우터와 레이어3 스위치는 복수의 네트워크를 연결.  
  논리구성도의 핵심은 몇 개의 네트워크가 어느 라우터와 레이어3 스위치로 연결되는지 파악하기 쉽게 정리하는 것.

- 물리구성도  
  각 기기의 물리적인 배치와 각 기기의 인터페이스가 어떻게 연결되어 있는지를 나타낸 구성도.  
  물리구성도의 핵심은 어느 기기의 어느 인터페이스가 어떠한 케이블로 배선되는지 파악하기 쉽게 정리하는 것.

하나의 물리구성도에 대응하는 논리구성도가 하나뿐이라고는 단정할 수 없음.  
네트워크 기기 설정에 따라, 같은 물리구성도에 대응하는 논리구성도가 많을 수 있음.
